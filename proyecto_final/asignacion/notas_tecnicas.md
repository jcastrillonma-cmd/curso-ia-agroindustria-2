# Notas técnicas para el proyecto final

## Objetivo

Esta carpeta se utilizará para alojar material técnico de apoyo para los estudiantes.

## Posibles archivos a incluir

- `tabular_base.py`
- `imagenes_base.py`
- `requirements_proyecto.txt`
- `instrucciones_entorno.md`

## Recomendación general

Los archivos base deben ser sencillos y servir como punto de partida, no como solución completa.

## Observación

El estudiante debe adaptar el código a su propio proyecto y documentar claramente cualquier cambio realizado.
