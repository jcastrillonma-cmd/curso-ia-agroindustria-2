# Proyecto final

En esta carpeta se encuentra toda la información del proyecto final del curso de Inteligencia Artificial para Agroindustria.

## Contenido

- `guia/enunciado_proyecto_final.md`
- `guia/entrega.md`
- `guia/rubrica.md`
- `datasets/README.md`
- `datasets/datasets_oficiales.md`

## Objetivo

Desarrollar un ejercicio aplicado de inteligencia artificial usando uno de los datasets oficiales del curso.

## Opciones del proyecto

- Clasificación tabular.
- Clasificación de imágenes.

## Tecnologías esperadas

- Python
- PyTorch o TensorFlow
- Git y GitHub
- LaTeX

## Nota

Los datasets y lineamientos oficiales del proyecto se encuentran en las subcarpetas correspondientes.
