# Dataset de imágenes

## Dataset base

PlantVillage

## Subconjunto oficial del curso

Se trabajará con un subconjunto de imágenes de hojas de papa, usando las siguientes clases:

- `Potato___Early_blight`
- `Potato___healthy`
- `Potato___Late_blight`

## Observaciones

- No se usará el dataset completo.
- El subconjunto será preparado por el profesor.
- El objetivo es desarrollar un clasificador de imágenes sencillo y reproducible.

## Nota

Esta carpeta documenta la modalidad de imágenes del proyecto final.
