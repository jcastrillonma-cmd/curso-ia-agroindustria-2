# Subconjunto de imágenes de papa

## Clases seleccionadas

El subconjunto oficial del curso para clasificación de imágenes está compuesto por tres clases:

- `Potato___Early_blight`
- `Potato___healthy`
- `Potato___Late_blight`

## Criterio de selección

El dataset original fue reducido para que el proyecto final sea manejable en tiempo, tamaño y dificultad.

## Balance del subconjunto

Se construyó un subconjunto balanceado tomando el mismo número de imágenes por clase.

## Uso esperado

El estudiante deberá usar este subconjunto para entrenar y evaluar un modelo sencillo de clasificación de imágenes.

## Nota

La versión exacta del subconjunto será organizada por el profesor y compartida como material oficial del curso.
