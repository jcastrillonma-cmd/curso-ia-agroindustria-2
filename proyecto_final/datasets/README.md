# Datasets del proyecto final

En esta carpeta se organiza la información de los datasets oficiales del proyecto final.

## Estructura

- `tabular/`: información y ejemplo del dataset tabular.
- `imagenes/`: información del subconjunto de imágenes.
- `datasets_oficiales.md`: descripción general de las dos modalidades.

## Modalidades disponibles

### 1. Modalidad tabular
Corresponde a un problema de clasificación binaria con variables numéricas.

### 2. Modalidad de imágenes
Corresponde a un problema de clasificación de imágenes de hojas de papa en tres clases.

## Recomendación

Los estudiantes deben usar únicamente los datasets y subconjuntos definidos oficialmente en el curso.
