# Datasets oficiales del proyecto final

En el proyecto final del curso, cada estudiante deberá trabajar con los **dos datasets oficiales** definidos para el desarrollo del trabajo:

1. un dataset tabular,
2. un dataset de imágenes.

Estos datasets corresponden a los dos componentes obligatorios del proyecto final.

---

## 1. Dataset tabular

### Ruta en el repositorio

```text
data/proyecto_final/tabular/phpnThNfi.arff
```

### Tipo de problema

Clasificación tabular.

### Descripción general

Este dataset deberá utilizarse para desarrollar el componente tabular del proyecto final.

El estudiante deberá:

- identificar las variables de entrada,
- identificar la variable objetivo,
- explorar la estructura general del dataset,
- preparar los datos para entrenamiento y evaluación,
- desarrollar un modelo de clasificación.

### Qué debe hacer el estudiante

Con este dataset, el estudiante deberá:

1. cargar y revisar los datos;
2. identificar entradas y salida;
3. preparar el conjunto de datos;
4. seleccionar un método de clasificación;
5. explicar brevemente la base matemática del método;
6. implementar la solución en **PyTorch** o **TensorFlow**;
7. entrenar el modelo;
8. evaluar los resultados;
9. interpretar el desempeño obtenido.

---

## 2. Dataset de imágenes

### Ruta en el repositorio

```text
data/proyecto_final/imagenes/potato_subset/
```

### Tipo de problema

Clasificación de imágenes.

### Clases incluidas

- `Potato___Early_blight`
- `Potato___healthy`
- `Potato___Late_blight`

### Descripción general

Este subconjunto de imágenes deberá utilizarse para desarrollar el componente de clasificación de imágenes del proyecto final.

El estudiante deberá:

- revisar la estructura de carpetas,
- identificar las clases,
- inspeccionar visualmente el conjunto de imágenes,
- preparar los datos para entrenamiento y evaluación,
- desarrollar un modelo de clasificación de imágenes.

### Qué debe hacer el estudiante

Con este dataset, el estudiante deberá:

1. revisar las clases y la estructura del conjunto;
2. preparar las imágenes;
3. seleccionar un método de clasificación;
4. explicar brevemente la base matemática del método;
5. implementar la solución en **PyTorch** o **TensorFlow**;
6. entrenar el modelo;
7. evaluar los resultados;
8. interpretar el desempeño obtenido.

---

## Uso de los datasets

Los estudiantes deberán trabajar únicamente con los datasets oficiales del curso.

No se debe escoger uno solo. El proyecto final exige desarrollar:

- el componente tabular,
- y el componente de imágenes.
